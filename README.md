# 🧬 Vitiligo Progression Monitor

## 📌 Problem Statement
Vitiligo is a skin condition where white patches appear due to loss of pigmentation. 
Tracking progression manually is difficult and inconsistent.

This project aims to **automate detection and progression analysis** using image processing and machine learning.

---

## 🚀 Solution
This system:
- Processes skin images using OpenCV
- Detects affected areas
- Calculates progression over time
- Uses ML techniques for prediction

---

## 🛠️ Tech Stack
- Python
- OpenCV
- NumPy
- Machine Learning

---

## ⚙️ How It Works
1. Input: Before & After skin images  
2. Image preprocessing (grayscale, thresholding)  
3. Detect white patches  
4. Calculate affected area percentage  
5. Measure progression speed  

---

## 📊 Output
- Affected skin area (%)
- Progression rate
- Visual detection of patches
